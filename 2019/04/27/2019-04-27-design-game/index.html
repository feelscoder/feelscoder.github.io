好玩又有趣，一起来测试你的设计能力

  • Larry
  • 2019-04-27
  • 工具资源
  • 像素测试

大家好，我是独立开发者 Larry ，经常有小伙伴微信私聊我，你的设计是怎么做的，如何提升自己的设计能力，UI设计有没有好的学习路径，大道理就是工欲善其事必先利其器——软件操作过关、多看、多收集、多想、多做 …… 在看和想的阶段，我推荐下面这几个游戏，一起来测测你的设计能力~

Can’t Unsee

网站地址：https://cantunsee.space

Can’t Unsee 是一款设计技巧游戏，网站提供教学关、简单关、中级关、困难关四个级别的测试挑战题来测试大家识别优秀设计的能力，其中教学关有 3 道测试题，在你选择你认为更正确或更恰当的设计后，网站会给出你的测试结果并解释该测试题的测试内容，同时你还可以按「比较按钮」或「Shift键」来对比测试题中给出的两个设计稿之间的差别。

初级、中级、困难这三个级别每一级别分别有18道测试题，测试内容包含 图标、字体、形状、间距、对齐、样式等等，随着级别的提升测试难度会逐步加大，尤其是在困难关，你会发现 UI 设计需要考虑的细节往往直接影响着一个设计的好坏。

评判哪个设计更好很多时候会受到评判者的主观因素影响，不过网站大部分的答案都是受到业内广泛接受的设计特征，如果有时间的话希望各位小伙伴都耐心测试一下吧～最后附上我的测试结果😂

SocialSizes

网站地址：https://socialsizes.io/

SocialSizes 是由 @assentorp 制作的一个关于我们常见社交平台的媒体图像和视频大小的网站，网站包含的社交软件有：Facebook、Instagram、LinkedIn、Twitter、Snapchat、YouTube、Pinterest 和 Twitch，并且提供了可供大家免费下载的 Sketch、Adobe XD 和 Photoshop 文件。

了解了这些尺寸，除了方便大家设计媒体图像或视频发布到对应的社交平台外，你也可以借鉴这些尺寸，运用的自己的APP设计中。

The Boolean Game

网站地址：https://boolean.method.ac

The Boolean Game 是一个有趣的布尔运算小游戏网站，网站提供一系列图标图形及构成图标的几何图形，将图形按照正确的顺序拖拽到右边的图标区域，然后再选择布尔运算——联合、减去、交叉、差异即可生成目标图标图形。

游戏分为联合、减去、交叉、差异、简易、中级和高级 7 个阶段难度依次递增，遇到不会的图标图形不用担心，网站会有提示，长按运算图标滑动还可以预览不同的运算结果，同时网站还提供了一系列操作音效，游戏过程体验非常棒！

Blobmaker

网站地址：https://www.blobmaker.app

Blobmaker 是一款能帮你快速生成不规则 svg 形状的在线工具，通过网站你可以随机生成形状，也可以修改形状颜色、复杂程度及对比度，简单实用。

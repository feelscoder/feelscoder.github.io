“阅后即焚” 的私密文件分享网盘服务

  • Larry
  • 2019-06-07
  • 工具资源

大家好，我是独立开发者Larry，今天是端午节，吃粽子是必不可少的，另外今天还是一年一度的高考，高考+端午=高粽，祝大家端午节快乐、心想事成 (///▽///)。

今天要给大家分享的是一个文件分享工具——Firefox Send，这个工具是由我们所熟知的Firefox火狐浏览器所属公司Mozilla推出的，一个开源私密文件分享服务工具，一起来看一下。

Firefox Send

网站地址：https://send.firefox.com/

Firefox Send 虽然是和Firefox火狐浏览器同属一家公司，但是并不只限于在火狐浏览器使用，Safari、Chrome、IE以及其他任何第三方浏览器都可以使用，使用时不需要额外下载或安装软件及扩充功能。

Firefox Send默认支持分享最大1GB的文件、1次下载、1天后分享链接过期，登录Firefox账户后，可以传送最大2.5GB文件、100次下载、 7天后分享链接过期。

拖放或选择要上传的文件，一次支持上传多个，总大小不超过限制即可。

之后你可以根据需要设置下载次数及文件过期时间，非登录用户上传文件仅支持1次下载，5分钟、1小时、1天保存时间，登录用户支持1次、2次、3次、4次、5次、20次、50次、100次下载，5分钟、1小时、1天、7天保存时间，分享链接会在文件保存时间到期后删除。

当然如果在文件保存时间未到期前，文件已达到下载次数上限，那么分享链接也会被删除。

在上传文件之前，你还可以手动勾选密码保护，并在本地手动设置下载密码保护增强文档安全。当然，无论你有没有设置密码保护，Firefox Send都会自动加密你的文档进行保护的。

文件上传过程会显示文档名称、文档大小、设定的下载次数、过期时长、上传进度，当然你也可以点击「取消」终止对该文档的上传及分享。

上传成功的文件Firefox Send会显示文件已加密，你可以直接复制文件链接，也可以点击「复制链接」按钮一键复制链接。

从Firefox Send下载文档时，如果文档有密码保护需要先输入该密码解锁文档，之后点击「下载」按钮就可以开始下载了，达到下载次数或设定时间后该链接就过期失效啦。

你上传的所有文档在失效前都会显示在上传文件框的右侧，你可以对这些文件进行下载、复制链接及删除操作。

Firefox Send目前已开源到GitHub，拥有7761个Star，感兴趣的话可以自己去看一看。

地址链接：https://github.com/mozilla/send

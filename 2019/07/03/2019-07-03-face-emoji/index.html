每个人都应该有一个酷酷的头像

  • Larry
  • 2019-07-03
  • 工具资源
  • 插画素材

大家好，我是独立开发者Larry～说到头像，每一个有社交需求的APP或网站都会涉及，无论是用户上传头像还是系统推荐头像，UI设计时大量的头像配图是必不可少的，如果一个一个从网站找配图不仅会花费大量时间，有可能还会涉及版权问题……

今天就给大家推荐一波头像资源，帮你快速完成头像配图。

UI Faces

网站地址：https://uifaces.co

UI Faces是一个聚合了大量真实人物照片的头像素材库，每张照片都经过Microsoft的Face API处理并标记年龄、性别、情感和头发颜色，从而提供更轻松的过滤和分类对头像素材进行筛选。

除了通过复制和粘贴使用头像素材外，UI Faces还提供了Sketch插件和Adobe XD插件，支持直接在Sketch及Adobe XD中生成头像。

Friendly Faces

网站地址：https://friendlyfaces.co

Friendly Faces的目标是构建一个具有包容性、多样性的头像制作工具，任何人都可以参与创建自己形象的个性化头像。

Friendly Faces支持随机生成一个头像，也支持手动更改五官、头发、服装、配饰等参数对头像进行调整，下载头像支持250px、140px和80px大中小三种尺寸PNG格式。

Avataaars Generator

网站地址：https://getavataaars.com

Avataaars Generator是由Pablo Stanley设计、Fang-Pen Lin开发的一款免费在线头像生成器，任何人都可以轻松制作出漂亮的个人头像！功能操作与Friendly Faces类似，提供随机生成头像或手动选择参数对头像进行调整，下载格式支持PNG、SVG、React代码及SVG IMG链接地址。

Avataaars Generator所有头像的原始Sketch文件均来源于Pablo Stanley设计的Avataaars网站，之前文章《最全面的App插画素材推荐》有推荐过。

Avataaars网站地址：https://avataaars.com/

Pictogrift

网站地址：https://luciorubeens.github.io/pictogrify

Pictogrify是一个开源库，支持从任何给定文本生成唯一的象形图。它是Github、Stack Overflow和其他论坛中使用的identicons的替代品，并且是一个更友好、有趣和完全可扩展的版本！

除了怪兽主体外，Pictogrify还预设了来自Avataaars库的男性及女性人物主题，很快还会有更多的形状和主题。

Personas by Draftbit

网站地址：https://personas.draftbit.com

Draftbit的Personas人物角色俏皮、现代、有趣，不依赖于性别、种族、年龄来限定角色，而是通过提供肤色、发型、胡须、服装、五官配饰、背景等参数来调整整体外观概念，当然Personas还支持随机生成一个头像，通过「Download Avatar」下载头像按钮可快速下载1024px大尺寸PNG文件，此外还可以下载Sketch文件或查看Github上的源代码。

Stubborn Generator

网站地址：https://stubborn.fun

Stubborn Generator是一套包含25个酷角色、50个背景物品和20个场景的免费插画图库，所有角色都可以使用Sketch符号和Figma组件进行编辑，下载素材库除Sketch及FIG格式外，还提供SVG和PNG格式，可以为你的项目快速创建一个独特的故事。

Diversity Avatars

网站地址：https://diversityavatars.com

Diversity Avatars是一个探索世界各地人们和文化的美丽、多样性头像素材库，目前拥有770个头像，包括各行各业、不同宗教信仰、不同国籍和种族。

Diversity Avatars提供30个免费头像供大家下载测试其产品质量，下载格式包括Ai、SVG、Sketch、Affinity Designe、Adobe XD、Figma、InVision Studio、FramerX和Iconjar等多种格式。

在扩张头像数量的同时，Diversity Avatars的头像质量也在持续优化，并且在所有1,000个填充轮廓头像发布后，网站计划将添加平面和普通轮廓版本，总共大约3,000个单独的头像。

Joe Schmoe

网站地址：https://joeschmoe.io

Joe Schmoes是由Jon＆Jess为开发人员和设计师设计创建的色彩鲜艳的角色库，可用作实时网站或设计模型的个人资料图片占位符，支持获取API链接地址或下载SVG格式，之前的文章《最全面的App插画素材推荐》也有介绍～

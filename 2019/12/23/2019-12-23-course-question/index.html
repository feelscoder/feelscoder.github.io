码力全开大学堂课程 - 常见问题

  • Larry
  • 2019-12-23
  • 课程
  • 码力全开

一. 码力全开大学堂的课程是什么形式呢？

  1. 码力全开大学堂的课程分为视频课程和专栏文章课程。
  2. 视频课程是由Larry研发和制作的录播视频课程。
  3. 视频课程的授课形式是录播+Demo源码+pdf资料，实战项目包含设计稿，购买后你可以自由安排自己的时间进行学习。
  4. 专栏课程的授课形式是文章+源文件+pdf资料。
  5. 视频的时长大多数是3-10分钟，没有废话，直入主题，讲解最有用的技术点。

二. 移动端课程包含的两个项目实战具体是什么？

  1. 从0到1的Swift和Objective-C课程都包含两个实战项目。
  2. 实战项目①是一个便签类项目，本地化存储数据（含iCloud云备份、暗黑模式等等）
  3. 实战项目②是一个带后台的项目，后台语言是PHP的TP框架提供API接口（数据库、接口设计都会有所讲解）
  4. 两个实战项目都包含完整的产品设计稿。

三. 码力全开大学堂的课程内容和时间安排是怎样呢？

  1. 课程的内容精心设计，在课程目录中可以看到详细介绍。
  2. 课程的时间安排，在课程概述的图文介绍中可以看到更新的频率。
  3. 用心做好每一个视频，让你的等待更有价值。

四. 码力全开大学堂的课程价格是一次性的吗？

  1. 课程标价就是整个课程的价格，购买后观看本课程内所有章节视频不作另外收费。
  2. 视频不支持下载，视频版权归码力全开所有，违者必究。

五. 这个课程需要学习多久？学习之后能达到什么水平？

  1. 多久能学完，和每个人的基础、接受能力和时间安排都有关，课程精心设计了合理的学习计划，安排16-20周（以周为单位），可按周计划学习，如学习的快，也可以提高学习速度。建议学习保持连续状态，这样学习效果更好。
  2. 课程学习完成后，可以达到独立开发产品的能力，提升职场竞争力。

六. 码力全开大学堂的课程有答疑吗？

  1. 购买后的学员可以加我的微信，然后我会拉入微信群，有问题可以在群内交流讨论或向我直接咨询。
  2. 在码力全开大学堂上的社区入口提交问题，我会及时进行解答，在这里交互问答，沟通交流。

七. 零基础非计算机专业能否学会课程？

零基础非计算机不是问题，重要的是你是否对编程有强烈的兴趣，兴趣是驱动坚持学习下去的动力，之前两期课程的学员有高中生拿到WWDC2018苹果开发者奖学金的，年仅16岁，也有北邮、厦大、兰州大学等在校的大学生以及从事互联网或其它行业的在职人员，现在码力全开大学堂的学习者年龄和职业更为广泛，加入码力全开大学堂我希望每位学习者都能用工匠精神不断雕琢自己的产品，追求完美和极致。

八. 码力全开大学堂的课程有观看有效期吗？

课程不限有效期，购买成功即可永久观看。

九. 码力全开大学堂的课程可以退款吗？

录播课程为虚拟商品，一经购买，不支持退款。

最后

在这里，帮你搭建一套完整的知识架构的视频学习网站，从设计界面、准备工具、学习语言、到构建App，内容一应俱全。

如果你对独立开发者、移动端工程师、设计师感兴趣，那么没有比码力全开大学堂更适合你的视频教程了，你可以在这里看录播视频、下载源代码、追踪学习进度、参加公开讨论以及更多。

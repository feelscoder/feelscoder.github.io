图片太小怎么办，无损放大神器来一波

  • Larry
  • 2019-03-27
  • 工具资源
  • 无损放大

大家好，我是独立开发者Larry，去年11月份的时候给大家推荐过一些图片压缩工具—「图片太大怎么办，压缩神器少不了」帮助解决图片过大的问题，相反的，有时候我们还会遇到一些素材图片过小、分辨率过低、放大模糊不清晰等问题，今天就给大家推荐7个图片放大工具，帮你解决这些问题。

waifu2x

网站：http://waifu2x.udp.jp/index.zh-CN.html

waifu2x 是一个支持中、英、日、韩等多国语言的在线图片放大工具，使用卷积神经网络对照片及动漫风格的图片或插画进行放大及降噪处理，可放大图像的最大尺寸是1500x1500px，可降噪图像的最大尺寸：3000x3000px，文件大小不超过5MB。

操作过程也很简单，选择将要放大的在线图片链接或者本地图片，然后选择风格、降噪程度及放大倍数、进行人机身份验证，最后转换在线预览或直接下载。

waifu2x 的 GitHub star 是 13871
GitHub地址：https://github.com/nagadomi/waifu2x

感兴趣的可以去看看。

bigjpg

网站：http://bigjpg.com

bigjpg 是使用最新人工智能深度学习技术——深度卷积神经网络（Deep Convolutional Neural Networks, based on waifu2x）对图像进行放大的工具，它会将噪点和锯齿的部分进行补充，实现图片的无损放大。对动漫、插画图片的放大几乎可以说是完美的，将小图片放大后，无论是色彩、细节、边缘，效果都很出色。同时也兼容普通的照片放大。

bigjpg 支持在线及离线操作，同时支持免费及付费使用，差别在于每月放大数量、放大速度、最大上传大小、最高放大倍数……具体如何选择就看你的使用需求啦～

Let’sEnhance.io

网址：https://letsenhance.io/cn

Let’sEnhance.io 是一款国外的在线图片放大工具，采用基于深褶积神经网络的尖端图像超分辨技术，实现大幅增加照片或图像的大小并且不失质量的效果。支持多种语言，需要用邮箱注册账户才能使用，并且免费用户只能免费使用5次，付费用户则可以体验更多功能及服务。

PhotoZoom

网址：http://www.photozoomchina.com

PhotoZoom包含 PhotoZoom Pro / Classic 两款图片放大产品，使用最新的S-Spline 技术 及 S-Spline XL 插值技术，拥有独家专利，自动调节、领先的差值算法等技术， 对不断放大的图片保持了极佳的品质，没有锯齿，不会失真，不过这两款产品都是收费的～

A Sharper Scaling

网址：http://www.photozoomchina.com

一个 windows 端的图片缩放工具，是开发者 Steffen 的一个业余时间项目，可以把照片缩放到最小50%最大400%，Windows操作系统32位，图片色彩通道8位并且仅支持彩色图像，工具具有一定的局限性，不过在图片的缩放效果上开发者做了很多的尝试，最终效果还是可以的。

reshade

网址：http://reshade.com/

reshade 也是 Windows 端的图片大小调整工具，能实现放大图像时没有通常的模糊、锯齿状边缘、晕圈、锐度损失或与图像放大相关的细节丢失，几乎可以在任何缩放系数下都能创建精确、清晰、逼真的视觉效果。

Imagus Chrome插件

网址：https://chrome.google.com/webstore/detail/imagus/immpkjjlgappgfkkfieppnmlhakdmaab?hl=zh-CN

Imagus 是一款可以通过将鼠标悬停在缩略图上来显示大图的 Chrome 插件，支持鼠标悬停链接或缩略图以显示图片、HTML5视频/音频和内容专辑等。

Imagus在显示大图的同时还支持快捷键对图片进行缩放、翻转、旋转、存储等编辑操作。

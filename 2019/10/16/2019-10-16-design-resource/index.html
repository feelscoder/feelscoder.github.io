Freebies.ByPeople

网站地址：https://freebies.bypeople.com/

Freebies.ByPeople 是面向设计师和开发人员的精选免费赠品集合，打开网站，默认显示所有允许商用的资源，关闭商业允许开关「 Commercial allowed 」可以查看供个人使用的非商用素材。

在 Freebies.ByPeople 你看到的每一个资源，网站都会给出它的原始页面链接地址，可以通过链接跳转到资源原地址下载资源，查看资源的授权许可信息将其用于个人或商业项目中。

在使用 Safari 和 Chrome 浏览器打开 Freebies.ByPeople 时，Freebies.ByPeople 首页看到的资源信息banner图右下角都有喜欢/原链接按钮图标，而在资源信息详情页面 Safari 打开后能看到资源标题上的banner图右下角显示喜欢/下载/原文链接按钮图标，Chrome 打开后不能找到下载资源或跳转资源原地址的链接图标，需要在 Freebies.ByPeople 首页来跳转资源原地址下载资源。

除了查看全部资源或者关键词搜索资源外，Freebies.ByPeople 首页还提供了字体、免费图标、设计赠品、图形主题、图形UI套件、HTML套件、HTML主题、代码插件、CSS代码段、电子书、3D模型、WP主题、WP插件、软件/ Web Apps 和交易等15个类别分类，我们逐一来看：

字体资源包含多种样式、精心选择和富有创意的免费字体，这些字体通常提供有OTF、TTF、EOT、WOFF 和 WOFF2等多种桌面和Web文件格式，方便使用。

免费图标_Free Icons

图标资源包括简约线性图标、面性图标、等距图标、拟物图标、单色图标、同色浓淡双音色图标、渐变图标、彩色图标等多种图标样式，大多数图标包提供EPS、AI、SVG、Sketch、Figma、XD、PSD、PNG等多种文件格式。

设计赠品_Design Freebies

设计赠品包含各种图形设计清单，比如：背景图案、渐变形状、矢量插画、设计模版、样机素材等大量启发创造力的设计资源，提供矢量或可编辑的PSD、AI、Sketch等多种文件格式。

图形主题_Graphic Themes

图形主题是一个适用于各种网站和应用程序的UI元素和屏幕工具包、页面模版库，包含的素材能帮助我们快速创建专业的项目组件，并且提供PSD、AI、XD、Sketch、Figma等多种文件格式。

图形UI套件_Graphic UI Kits

图形UI套件是用户界面UI套件的汇编，包括针对Web、应用程序、桌面和移动项目的UI包，提供Sketch、Figma、XD、AI和PSD等多种文件格式。

HTML套件_HTML Kits

HTML套件是多种帮你集成到桌面、Web和移动应用程序项目的免费HTML/CSS UI工具包，其中一些是使用流行的库和框架创建的，比如Bootstrap，Materialize。

HTML主题_HTML Themes

HTML主题涵盖了登录页面、皮肤主题、网站模版、Web和移动应用程序等多种易于下载的HTML/CSS主题和模版，其中大多数都是基于Bootstrap的。

代码插件_Code Plugins

代码插件是为一些编程工具，比如Javascript、Vue、React、Angular、CSS和CSS预处理器、WordPress、Ruby等精心挑选的插件、库、框架，帮你构建你的项目。

CSS代码段_CSS Snippets

CSS代码段包含大量的CSS示例、实用程序、有用的代码和工具，这些工具利用了各种预处理器，比如：SASS、LESS、Stylus、PostCSS和CSS库等，可以用作启发或在你的项目中包含和扩展。

电子书_E-Books

电子书包含大量精美的汇编书籍供大家下载，比如：图形和网页设计、编程和网页开发（包括前端和后端）的电子书、完整指南、汇编和教程、理论和实践课程等，帮助你获得促进项目发展所需要的知识。

3D模型_3D Models

3D模型包含对象、场景和完整场景，并且提供 FBX、CD4、3DS、MAX、Blend等多种文件格式，功能实用、具有启发性，是室内设计的有用的参考素材和头脑风暴工具。

WP主题_WP Themes

WP主题包含大量精心挑选的免费WordPress主题列表，这些主题针对许多目的，例如博客、企业、慈善机构、投资组合、体育、房地产、旅行等等，你可以找到可用于个人和商业用途的最新版本。

WP插件_WP Plugins

WP插件是精选的最新版本WordPress插件，比如：Builders、SEO和图像优化、自定义字段管理、安全性、表单和调查等等，所有这些你都可以免费下载并用于商业用途。

软件 / Web Apps_Software / Web Apps

软件/ Web Apps是包含大量工具、桌面应用程序、框架、管理平台以及完整的成品软件实用程序的目录，可以帮你进行团队工作或创建桌面、Web和移动应用程序。

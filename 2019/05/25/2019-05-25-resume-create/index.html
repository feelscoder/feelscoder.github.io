换一种方式创建一个更好的简历

  • Larry
  • 2019-05-25
  • 工具资源
  • 简历工具

大家好，我是独立开发者Larry，时间总是过的很快，转眼马上就六月份了，各大高校的毕业展已经陆续开启，毕业的日子也是越来越近，总之，又有一大波小伙伴要开启他们的职业之路了……

无论是毕业生还是职场人，只要你还需要工作，简历总是必不可少的，通常大家是如何制作简历的呢？打开Word，设计表格，将文本放入表格，设计样式，调整布局，更改样式，更改布局……太麻烦了！所以今天我给大家带来了一个超级棒的在线简历生成器工具——FlowCV，让你换种方式轻松创建一个更好的简历。

FlowCV

网站地址：https://flowcv.io

FlowCV是由创作者Lauris和Alex为我们搭建的一个简洁可编辑的免费现代简历生成器，提供以可读的方式输入文本、轻松将简历包装成专业设计效果、实时预览以及快速下载等功能，并且这一切都是完全免费的。点击「Try It」按钮，我们一起开始体验一下吧～

进入FlowCV简历制作，首先看到的是页面右侧引导文字描述及相关说明，左侧是操作功能及步骤入口，包括内容、设计及检查，接下来我们一一体验。

内容

内容模块当然是对简历内容的录入，左侧可按步骤提示输入内容，右侧模块实时显示已录入的内容，同时支持实时保存及下载。

第一步输入姓名之后，进入个人资料的录入，并且页面提示添加更多内容，先看一下个人资料编辑：

左侧信息录入，就像是在注册一个网站一样，内容清晰且直观，体验也很棒，打开图片上传编辑，你就能确切感受创作者的考虑有多么细致～

关于图片的尺寸，你可以选择竖的、方的、圆的，关于图片大小你也可以拖动放大，并且支持顺时针、逆时针旋转，即使这个头像只是占据简历的一小部分，网站也是将其功能做的非常全面和完善啦。

关于个人介绍，自己制作简历的时候，包含姓名、头像、电话、邮箱、位置基本已经够全面的了，其实你还可以更完善，在保存按钮的上面还有一个“more”更多，点开之后支持你录入更多个人信息，比如网站，这个对于想要更多表现自我的职业来说，真的非常非常必要。

编辑完个人信息，网站继续提示你添加更多内容，对于创作简历，相信网站比你想象的更全面和强大：教育经历、技能特长、专业经验、证书奖项……并且你还可以自定义发布你想要展现的内容。

在你录入简历内容的时候，不知道有没有留意到「Tips」按钮，是的，这是开发者为大家提供的又一惊喜——有关优化简历内容的一系列建议，帮你规避简历制作中常见的陷阱。

设计

内容部分录入完成之后，你就可以开始对其进行视觉设计优化了，网站提供了字体、颜色、布局、间距等多方面的设计参数调整，根据自己的审美确定简历的最终设计效果，整个设计调整只需要你敲动鼠标就可以快速实现，并且效果都很精美。

检查

如果你在进入网站后一直没有登录网站，检查阶段网站会提醒你创建一个账号，并且会优先使用你在简历填写阶段输入的邮箱进行注册，当然你也可以使用其他邮箱注册，登录之后在检查页面，网站会给你提供一个链接，供你分享查看并收集反馈。

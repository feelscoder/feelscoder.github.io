这些UI套件让你轻松设计出好产品

  • Larry
  • 2020-01-02
  • 工具资源
  • 设计资源

大家好，我是独立开发者Larry～，2020年的第一次干货，我们一起来看一看UI套件，今天推荐两个UI套件，一个免费一个付费，质量还不错。

这些组件具有流畅、出色和吸引人的设计，我们以强大的UI基础开始您的项目，以节省您的时间和金钱。

Stark UI Kit

网站地址：https://www.baianat.com/resources/stark

打开网站可以看到免费下载的UI套件入口，下面的效果图就是UI套件中的一部分效果图。

可以自定义Stark UI Kit，并将其适当地用于许多领域，包括商业，广告，电子商务，新闻社等。

Stark UI Kit包含许多UI组件，这些组件具有流畅、出色和吸引人的设计，而且可以定制以满足您的所有需求。


Stark UI Kit的开发具有适合所有拉丁语言（包括阿拉伯语）的功能，这就是真正的普遍性。

Stark UI Kit中使用了多种颜色，以使UI外观生动适合不同的口味，同时Stark UI Kit由128个平面彩色图标＃[br]组成，适用于许多不同的用途。

Stark UI Kit支持浅色和暗黑两种的设计风格。


Root UI Kit

网站地址：https://rootwireframekit.com/?ref=prototyprio

线框和设计入门套件，具有出色的设计基础->根UI套件，设计出更好的线框和产品，同时根目录包括功能资源，这些功能具有逻辑组织的样式和调整大小的约束条件，以加快工作速度。

8个类别中的120个布局，更快地创建原型，享受组织完善的库，可帮助您快速创建线框和网站布局。


轻松自定义您的线框，利用最少的样式创建各种精美的线框，8种颜色、4种文字样式、无障碍对比。

使自己能够创建友好的用户流，并向同事和客户更好地展示您的想法。

  1. 以强大的UI基础开始您的项目，以节省您的时间和金钱。
  2. 带有整齐有序的组件覆盖，可快速从线框过渡到最终设计
  3. 可调整大小的元素和智能布局功能可加速您的工作流程。
  4. 通过专门设计的24px图标集和免费的Google字体来丰富线框。

使用 Root UI Kit 增强线框工作流程，付款一次获得免费更新，无限次数使用。

PS：公众号后台回复「UIKit」即可免费下载Stark UI Kit套件资源。
